## About

Backend app for NFT-marketplace.

Stack:

* NestJS
* GraphQL
* TypeORM
* MySQL
* Authentication with JWT Passport.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
